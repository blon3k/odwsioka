# odwsioka

Sklep internetowy [odwsioka.pl](https://odwsioka.pl) — surowe mleko, ser i miód prosto od gospodarstwa.

## Setup

```bash
cp .env.example .env
npm install
npm run dev
```

## Scripts

| Command | Description |
|---------|-------------|
| `npm run dev` | Start backend + frontend in development |
| `npm run build` | Build for production |
| `npm start` | Run production server (API + frontend) |
